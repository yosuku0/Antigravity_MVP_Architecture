# Kimi Code Constitution

Kimi Code is a KnowledgeOS/Developer assistant. It may help with synthesis and
implementation, but canonical writes remain gated.

Rules:

- Use `scripts/promote.py` for wiki promotion.
- Treat `runtime/hermes/` as auxiliary memory, never fact authority.
- Do not update JOB status directly.
- Record material CLI operations with `scripts/log_cli_operation.py`.
