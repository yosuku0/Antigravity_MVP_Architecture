# Antigravity MVP Architecture

NIM-Kinetic Meta-Agent MVP.

## Python Version

This MVP requires Python 3.12 for CrewAI compatibility.
If your system has Python 3.14+, use `py -3.12` or create a venv with Python 3.12.

```bash
python -m venv venv
# On Windows with multiple Python versions:
py -3.12 -m venv venv
.\venv\Scripts\activate
pip install -r requirements.txt
```
